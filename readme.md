# git

## 01 配置

```
git config --global user name "your-naem"

git config --global user email "your-email"

ssh-keygen-t rsa -C "your-email"

```

## 02 本地初始化

```
git init

# 与远程库连接
git remote add origin <git地址>

```

## 03 上传

```
# 先拉取
git pull
git pull origin master

# 添加更改到本地库
git add .
git add a.txt

# 填写提交信息
git commit -m "修改信息"

# 最后推，提交
git push
```
